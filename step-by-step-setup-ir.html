$ sudo chown vagrant:www-data /var/www/html/drupal/web/libraries
$ drush wfld
$ drush webform:libraries:composer > /var/www/html/drupal/composer.libraries.json

Add to the composer “extra: {” section

        "merge-plugin": {
            "include": [
                "composer.libraries.json"
            ]
        },

Then to trigger composer to copy the files. $ composer update --lock

Add Webform

admin > structure > Webforms > build > elements

In the current Bepress submission portal a student gets a form with elements to fill in.

Click + Add Element and select the type of elements needed.

After selecting the type and filling out the form the element can be made a conditional element. In this example the last option in this list is going to be a dropdown list of Degree Names. The only names that should show for a Thesis Degree Type should be Masters degree names. This option is found under the Conditions tab and the elements will auto fill your Value.

The arrangement of these elements are drag and drop.

Clicking Source will give a yaml file that can be imported into a different maching development vs production servers making the development cycle that much easier.

There are numerous other features that had been utilized in TRACE. There is a Email / Handlers tab to allows additional actions and behaviors to be processed when a webform or submission is created, updated, or deleted. Handlers are used to route submitted data to external applications and send notifications & confirmations.

Permissions can be set for the form submissions with the Webforms modules and within the separate Drupal’s permissions section.
